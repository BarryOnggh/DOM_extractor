# Security and responsible-use policy

This repository is a fictional, local-only test fixture for browser scam-detection research. I created this fictional scam-website fixture to test my browser extension's scam-detection capabilities.

## Reporting a problem

If you identify a safety issue in this fixture, open a private issue with the repository owner. Do not include real credentials, identity numbers, payment details, one-time passcodes, victim information, or links to live phishing pages.

## Out of scope

This project must not be modified or deployed to:

- impersonate a real organisation, public service, bank, or person;
- capture, transmit, or store sensitive data;
- distribute software, documents, or malware;
- host active phishing content; or
- target, test, or contact people without authorisation.

All security testing should stay in an authorised local or isolated environment.
